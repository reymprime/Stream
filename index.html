Application Specifications

Gnokz Pro is a premium, optimized, and seamless media client specifically engineered for a streamlined listening experience of Music and Podcasts. It fully supports cloud link streaming and localized storage integration to deliver flawless audio playback wherever you go.

Creator Reymark Delagao
App Type Podcast & Music
UI Framework Google Antigravity
Code Editor Spck Editor
Cloud Sync Platform Firebase Firestore System
User Guide & Documentation:
  • Tap the (+) button above to paste and import any YouTube audio or podcast link directly into your library.
  • Navigate to the Offline tab to upload and stream local .mp3 or .m4a files directly from your system storage.
  • Tap the Download Icon on cloud-sourced items to easily convert and save your favorite media using our integrated download portal.
